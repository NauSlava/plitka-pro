# Проект "nauslava/rubber-tile-lora-v45"

## Описание
Это полная структура проекта модели "nauslava/rubber-tile-lora-v45", которая использует НАШУ обученную модель для генерации изображений резиновой плитки.

## Ключевые Особенности
- **ОБУЧЕННАЯ МОДЕЛЬ:** Использует наши LoRA и Textual Inversion файлы
- **Простой API:** WEB-интерфейс с автоматическим преобразованием промптов
- **Высокое качество:** Максимальное качество благодаря обученной модели
- **Оптимизация:** Эффективное использование GPU ресурсов

## Быстрый Старт
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите модель: `cog predict`
3. Используйте API для генерации изображений

## Технические Детали
- **Базовая модель:** Stable Diffusion XL 1.0
- **LoRA:** Наши обученные адаптеры (сила 0.7)
- **Textual Inversion:** Наши обученные эмбеддинги
- **Токены:** `<s0><s1>` для активации модели
- **Формат:** safetensors для безопасности
